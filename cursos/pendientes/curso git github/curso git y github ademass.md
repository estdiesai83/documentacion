**Git** es una herramienta de **control de versiones** gratuita y de código abierto que los desarrolladores instalan localmente en sus ordenadores para realizar el seguimiento de los cambios,, mientras que **GitHub** es un servicio en línea diseñado para ejecutar Git en la nube, donde puedes almacenar, compartir y trabajar junto con otros usuarios para escribir código.

**Git** proporciona herramientas para aislar los cambios y volver a combinarlos posteriormente. Las ramas, que son punteros ligeros para el trabajo en curso, administran esta separación. Una vez finalizado el trabajo creado en una rama, se puede combinar de nuevo en la rama principal (o troncal) del equipo.
