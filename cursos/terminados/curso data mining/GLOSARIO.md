- Analisis de sentimiento: por ejemplo se utiliza para minar datos de redes sociales (facebook, ...) que utilizan para buscar terroristas,..
- ciberpatrulla: es un poli que tiene un canal para hacer búsquedas (buscador de datos)
- CIS: Centro de Investigación Sociologica
- **CRM**: Customer Relationship Management  gestión o administración de relaciones con el cliente
- Engagement: es como una fidelizacion (buscar info) https://blog.hubspot.es/marketing/guia-engagement
- ETL: (Export, Transform, Load) extracción, transformación o carga. proceso que permite a las organizaciones mover datos desde múltiples fuentes, reformatearlos y limpiarlos, y cargarlos en otra base de datos, data mart, o data warehouse para analizar, o en otro sistema operacional para apoyar un proceso de negocio.
- IAs:
- InteligenceX: motor de búsqueda parecido a google utiliza palabras con sus etiquetas, tiene buscadores por ejemplo para matrículas, números de teléfono
- KDD: (Knowledge Discovery in Databases) proceso utilizado para llevar a cabo la extracción automatizada de conocimiento partiendo de grandes volúmenes de datos. Etapas del proceso kdd: **Preprocesamiento/limpieza.** **Transformación/reducción.** **Minería de datos (data mining).** **Interpretación/evaluación**.
- KPI: (key performance indicator) (indicador clave de rendimiento), **es una métrica cuantitativa que muestra cómo tu equipo o empresa progresa hacia tus objetivos empresariales más importantes**. **Puede ser un número o un porcentaje.** https://es.wikipedia.org/wiki/Indicador_clave_de_rendimiento

- IDS: Sistemas de Detección de Intrusos, es un programa de detección de accesos no autorizados a un computador o a una red. El IDS suele tener sensores virtuales con los que el núcleo del IDS puede obtener datos externos. 

  IDS: https://suricata.io/ o https://www.snort.org/downloads

- IPS:  Sistemas de Prevención de Intrusos

  Ambos sistemas (IDS e IPS) se encargan de vigilar el tráfico, y para ello examinan la red y los puertos, analizando paquetes de datos, para detectar patrones sospechosos.
