## Cestas Louvers (persianas)

#### 1. Domo/Louvers/changeZoneLouversPosition

Cambia la posicion de las persianas de una zona. 

```php
data:
1. id (id zona)
2. value (1-> UP, 1-> DOWN,3 -> STOP, 4->FAVOURITE)
```



#### 2. Domo/Louvers/changeZoneLouversDim

Cambia la oscuridad de las persianas de una zona. 

```php
data:
1. id (id zona)
2. status (0 -> +, 1-> -)
```



#### 3. Domo/Louvers/changeLoungeLouversPosition

Cambia la posicion de las persianas de una estancia. 

```php
data:
1. id (id estancia)
2. value (1-> UP, 1-> DOWN,3 -> STOP, 4->FAVOURITE)
```



#### 4. Domo/Louvers/changeLoungeLouversDim

Cambia la oscuridad de las persianas de una estancia. 

```php
data:
1. id (id estancia)
2. status (0 -> +, 1-> -)
```
