## Cestas Blinds (estores)

#### 1. Domo/Blinds/changeZoneBlindPosition

Cambia la posicion del estor de una zona. 

```php
data:
1. id (id zona)
2. value (1-> UP, 0-> DOWN)
```



#### 2. Domo/Blind/changeZoneBlindValue

Cambia el nivel del estor de una zona. 

```php
data:
1. id (id zona)
2. value (valor 0-100)
```



#### 3. Domo/Blinds/changeLoungeBlindPosition

Cambia la posicion de los estor de una zona. 

```php
data:
1. id (id estancia)
2. value (1-> UP, 0-> DOWN)
```



#### 4. Domo/Blind/changeLoungeBlindValue

Cambia el nivel de los estores de una estancia. 

```php
data:
1. id (id estancia)
2. value (valor 0-100)
```
